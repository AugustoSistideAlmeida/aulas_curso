Estudo de fontes

A altura de uma fonte é definida pela letra X minúscula, todas as outras letras obedecerão a altura x, isto é, terão todas o mesmo tamanho da letra x minúsculas.

As letras maiúsculas obedecem o espaçamento chamado de tamanho das maiúsculas, mantendo um tamanho de letra maiúscula padrão.

O vazamento de letras minúsculas, como a perna do b minúsculo, costumeiramente segue o mesmo tamanho das fontes maiúsculas, mas não é um padrão sempre seguido.

Assim o que é “vazado” para cima das letras minúsculas, é chamado de ascendente, e o que é vazado para baixo da altura x das letras é chamado de descendente.

O espaço total de uma fonte, somando descendente, altura x, ascendente, até o topo de uma letra maiúscula, é chamado de corpo.


Serifa: pequeno traço remanescente na ponta das letras de uma fonte.Serve para formação de linha imaginária e facilitar a identificação de linha e facilitar a leitura de um texto.


Haste: Parte retangular horizontal ou principal de uma letra.


Filete: Parte que se assemelha a haste, mas é horizonta.


Arco: Parte curva da letra.
